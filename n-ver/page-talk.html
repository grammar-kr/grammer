talk [ 속내-표출 ]           품은 바대로 (말이)펼쳐지게 하다

http://www.grammar.kr : 다음사전
글자크기

영어 구문에 빈번히 쓰이는 말 동사에는 speak, tell, talk, say, 등등이 있다.
그 중에 talk 는 어떤 개념으로 작용하는 것일까.




①  Peace  talks  had to be aborted .
  ( 평화회담은 중단되어야 한다)
②  They  talked  for hours and hours .
  ( 그들은 몇 시간 동안 계속 이야기했다)


① 의 구문에서 talk 는
명사로 쓰였고, 의미는 회담(토론)이다.

반면에 ② 구문에서는 말 동작으로 쓰였지만
해석된 의미가 '이야기 하다' 가 될 수 있는 것을 보면

talk 를
단지 물리적 입놀림의 소리 동작만으로는 이해할 수 없게 된다.




③  Why don't you  talk  black into white .
  ( 나빠도 좋다고 우기세요 )
④  I  talk  in my sleep .
  ( 나는 잠꼬대를 한다 )


③ 구문에서의 해석을 보면
어떤 '우김질' 을 하는 말 동작에 talk 동사가 선택되었는데

④ 구문에서
'잠꼬대' 의 말 동작에도 선택되는 것을 어떻게 이해해야 하는 것일까.




⑤  The kettle is  talking  on the stove .
  ( 난로 위에서 주전자가 소리 내며 끓고 있다 )

⑥  He  talked  sailor .          
  ( 그는 뱃사람의 말투로 이야기했다 )


⑤ 의 구문에서
주전자의 물 끓는 소리가 talk 로 표현될 수 있고

⑥ 의 구문처럼 '목적어' 위치의 sailor 가
talk 의 동작 속성을 결정해 주는 현상을 이해하며

talk 동사가 보여주는 역할기능과 그에 따른 작용개념을 도출해보자.
1. Are you going to have a talk?
( 대화를 할 거니? )

talk 는 말(소리)로 행해지는 동작 어휘이다.

이 구문에서는 말로 행해진 동작 상황이 형상화(명사화)되어
(-a 의 역할기능에 의해)으로 제시되면서

말의 오고감, 즉 '대화' 라는 의미로 쓰였는데
'대화' 는 상대를 필요로 한다.

그렇다면 어휘 talk 는
대화 상대를 직접 대상으로 다루는 동작 속성으로 작용하는 것일까.




①  Thank you for  talking  to me .
  ( 내게 말해줘서 고마워 )


화자가 대화 상대방인 you 를 앞에 두고서

you 가 자기(to me)에게 talking 했음을 고마워하는 이 표현에서
talk 는 대화의 상대인 me 를 to 로 지정하는 형태로 쓰였다.

분석을 위해
이 구문의 동작 상황을 평서문화 한 심층적 구문구조를 가정해보자.




②  You  talk  to me .


③  You  talk   yourself   +   yourself   is talked  +   it is   to me 
                ⓐ                                   ⓑ                 ⓒ
     ( 너는 말하게 한다 )     ( 너 자신은 말하게 된다 )   ( 그것은 me 에게 이다 )



이와 같은 분석은, 심층 구조상으로
동사를 반드시 목적 대상에게 동작을 끼치게 하는 '사역성' 으로 타동사화 하고

주절 주어의 능동적 행위(ⓐ)에 따른 피역 대상의 수동(피동)적 처해짐(ⓑ)과
그것이 처한 상황에 대한 부가 서술(ⓒ)이

영어 구문 형성의 기저 현상임을 받아들이고 시도한 것으로써
이것은 다음과 같이 정리된다.


주어의 능동적 행위에 따른 의도대상의 수동(피동)화 현상이 영어구문 형성의 심층
기반에 있고 '주어 + 동사 + 대상(심층적 피동대상) + 상황구체화' 형식으로 나열된다.


③ 구조와 같은 심층적 발화 메커니즘을 적용해보면

구문 ② 의 talk 동작은 주어가 스스로에게 사역하는 꼴이고
그 언어(talk) 전개 사역은 (to) me 에게 행해진 것으로 구체화 된다.

본문 1. 에서는 a talk 에 대한 해석어가 '대화' 인데
이 해석어에 따르면 상대와의 '말 주고-받음' 을 생각해야 하지만

예문 ① 의 우리말 해석을 보면 '말해줘서' 라는 해석어가 자연스러우므로
talk 는 일방적 말씀전개를 표현했다고 볼 수 있다.




④  Did you  talk  to your teacher ?
  ( 선생님에게 말씀 드렸니? )


④ 의 표현에서 보면, 대상인 선생님과 you 가
꼭 대화를 나누지 않았을 상황으로도 인지할 수 있기 때문에

동사 talk 의 동작 속성을 주어의 '의사 표출' 로 보고
다른 전치사로의 상황 구체화에서 살펴보자.
2. What are you talking about?
( 무슨 말을 하는 거야? )

본문 2. 와 같은 표현은
말하는 사람이 무슨 말을 하는지 알 수 없을 때 듣는 사람이 흔히 하는 말이다.

앞의 1. 본문에서
talk 의 동작 속성이 대화 대상을 '직접목적어' 로 취하지 않음을 보았는데

본문 2. 에서도 보면, talk 의 동작 속성상
다루어야 할 주제가 되는 대상 또한 직접목적어 형태로 다루지 않고 있다.




①  You are  talking  about what .
  ( 너는 뭔가에 대해 말하고 있다 )


②  You  talk   yourself   +   yourself   is talked   +   it is   about what 
               ⓐ                                    ⓑ                     ⓒ
     ( 너는 말하게 한다 )     ( 너 자신은 말하게 된다 )   ( 그것은 what 에 관한 것이다 )



talk 동사가 대상으로 다루는 것은 what 이지만
talk 를 하는 대화 대상은 아니다.

그렇다고 대화의 주제가 되는 직접 대상도 아니다.
what 은 about 의 대상이다.

이 전치사 about 의 에 따르면
what 을 둘러싸고 있는 이것저것에 관한, 즉

what 의 내용을 하나로 말하지 못하고 포괄해서 인지하게 하는
about 상태로의 주제를 대상으로 다루고 있다.

이런 표현 형태로 알 수 있는 것은

talk 동작 속성이 주어 자신만의 생각들을 겉으로 표출하는
'속내 표출' 로 되어 있다는 것이고

talk 가 지닌 이 같은 어휘 속성 때문에
어떤 주제를 두고 벌이는 '토론형의 말 동작' 동사로 작용한다고 간주할 수 있다.
3. I will talk to the department head for you.
( 학과장님께 말씀드려 보겠습니다 )

본문 3. 의 표현에서
talk 가 다루는 주제가 되는 대상은 알 수 없다.

무엇을 말하겠다는 것인지는 알 수 없으나, for you 를 통해서
주어(화자)가 you 를 하는 말(talk)인 점만은 알 수 있다.

그렇다면, talk 동작이 다루게 될 주제(내용)는
화자가 생각하는, 즉 내포하고 있는 간직된 속마음이고

talk 는
그 속내를 곁으로 드러내는 말씀 전개인 셈이다.




①  I went on  talking  even though no one was listening .
  ( 듣는 사람 없어도 말을 계속했다 )


① 의 구문도
talk 의 대화 대상도, 주제 대상도 없이 말씀 전개 상황만을 표현해주고 있다.

주어가 마음속에 간직한 어떤 것을 말씀으로 펼쳐내는 상황을 그려 볼 수 있고
다음 ② 의 예문도 그렇게 인지하게 해준다.




②  I  talk  in my sleep . ( 잠꼬대를 한다 )



꿈과 잠꼬대는 '잠재의식의 발로' 라고 한다.

마음속에 담아 둔
무의식 속의 어떤 것들이 표현되는 현상이라고 보면

잠꼬대 동작을 표현한 talk 의 동작 속성이
'속내 표출의 말씀 전개' 라는 역할기능으로 되어 있음을 잘 보여주고 있다.
4. Now you're talking!
( 이제야 말이 통하는 군 : 그렇게 나와야지 )

본문 4. 는
단순히 주고받는 '말동작' 을 하고 있다는 표현이 아니다.

대화 상대가 비로소
화자의 의도를 파악하고 속마음을 펴보였다는 뉘앙스 구문이다.

본문 4. 표현에서도
talk 동작은 주어의 마음속 '품은 바' 가 표출되는 말씀 전개임이 확실해진다.

다음 ① 구문은
'품은 바' 가 확장되어 적용되는 예일 수 있다.




①  The kettle is  talking  on the stove .
  ( 난로 위의 주전자가 끓고 있다 )


주전자가 말을 할 리 없다.
단지 끓는 주전자가 내품는 수증기에 의해서 소리가 날 뿐이다.

이 소리 펼쳐짐에 talk 가 선택될 때
'품은 바' 가 전제로 작용했을 것이다.

수증기는 '품은 바' 의 대상이고
그 수증기의 표출 소음은 '소리 펼쳐짐' 으로 드러냈다고 볼 수 있기 때문이다.




②  He  talked  sailor . ( 그는 뱃사람 말투로 말한다 )



② 의 표현에서 He 가
sailor 를 '듣는 대상' 삼아 말(talk)하는 것도, '주제 대상' 삼아 말하는 것도 아니다.

talk 는 대상을 직접다루지 않기 때문인데

우리말 해석에 따르면
뱃사람(sailor)은 talk 동작에 전제적으로 작용하는 '품은 바' 일 수 밖에 없다.

다시 말해서
몸에 베인 뱃사람식의 말투가 품은바 되어 말씀 전개시 표출되어 나옴을 말한 것이다.
5. I tried to talk her out of leaving.
( 그녀에게 떠나지 않도록 설득하려고 애썼다 )

앞의 장에서 동사
talk 는 대상(-목적어-)을 직접 다루지 않는 것으로 보았다.

그렇지만, 이 본문 5. 의 경우를 보면
형태적으로는 talk 의 대상이 전치사 없이 주어져 있다.

달리 보면 의 구문처럼 목적어로 주어졌으나
목적보어 없이는 성립되지 않는 대상(-불안전목적어-)이게 된다.

이런 현상을 지금까지 도출한 바에 적용해보면

her 는 '품은 바' 속의 상황 주어이지
결코 talk 로의 대화상대로 선택한 것이 아니어야 한다.

다시 말해서 talk 에 의한 '말씀 전개' 의 펼쳐짐을 보게 되는,
떠나지 않기를 설득당하는 마음속에 의도한 대상이게 된다.


지금까지의 분석에서 드러난
talk 의 역할기능과 그에 따른 작용개념을 정리하면 이렇게 된다.



첫째, talk 는 '의사 표현' 의 동작 동사이다.

둘째, talk 는 '품은 바' 를 전제로 동작한다.

셋째, talk 는 (마음) 속에 간직 된 것을 표면화하는 '속내 표출' 동작이다.


역할기능 : 속내-표출
작용개념 : 품은 바대로 (말이) 펼쳐지게 하다
