set [ 굳힘-진입 ]                    굳혀짐에 들어서게 하다 http://www.grammar.kr : 다음사전

글자크기

set 은 동사로 작용하면, .. 설치하다, .. 놓아두다, 등으로 우리말화 할 수 있어서
'갖추어 두다' 정도로 동작을 한다고 보면 될 듯싶다.




①  Set  the alarm for 5 o’clock .
( 알람을 5시에 맞춰 놔라 )
②  Everything is  set .
( 모든 것이 세팅되어졌다 )


① ② 는
설치 행위에 대한 갖춤의 상태를 표현했다고 할 수 있다.

그렇지만
③ 과 ④ 의 경우는 다르게 다가온다.




③  He  set  fire to the house .
( 그는 집에 불을 질렀다 )
④  The hijackers  set  the hostages free .
( 납치범들이 인질들을 풀어 주었다 )


③ ④ 두 구문에서 set 동사의 발화 의도는
'설치' 나 '놓음' 의 갖춤 상태 표현이 아니다.

표적 대상이
어떤 상황에 처해지게 만드는 착수 동작을 한다.

그뿐 아니라
다음의 경우는 굳혀짐의 상태를 말하기도 한다.




⑤  The surgeon  set  her broken arm .
( 의사는 그녀의 부러진 팔을 접합시켰다 )
⑥  Schubert  set  many poems to music .
( 슈베르트는 많은 시에 곡을 붙였다 )


set 동사 동작이 드러내는 이런 의미 표현들을 살펴보며
하나의 어휘에 하나의 개념이 작용하고 있다고 보고

제시되는 구문들을 분석해가며
set 동사의 심층적 역할기능과 그에 따른 작용개념을 도출해보자.
1. All set!  let's go.
( 준비 완료. 갑시다 )

본문 1. 에서의 All set 은 '준비 완료' 로 번역되어 있다.

준비가 다 된, 즉 뭔가를 하고자 할 때
미리 갖추어야 할 것들의 완성된 상태 표현에 set 이 쓰였을 것이므로

이 set 은 '갖춤' 을 말한 것이고
다음 ① 과 ② 구문은 그러한 갖춤의 상태 표현을 잘 드러내주고 있다.




①  Everything is  set .
( 모든 것이 세팅되어 있다 )
②  The table was  set  for six guests .
( 식탁은 손님 6명이 앉도록 차려 졌다 )


set 은 우리의 언어생활에서도
셋팅(setting)으로 외래어화 된 듯 친숙해져서 그다지 어려움 없이 인지하게 되는데

어떤 것들의 설치나 갖추어진 구성품들의 집합체를 지칭할 때
이 어휘가 선택됨을 볼 수 있다.

그렇지만
이 개념을 모든 set 어휘 사용처에 적용시킬 수는 없다.
2. He set his face to the work.
( 그는 그 일에 착수했다 )

본문 2. 에서의 set 을
우리말 '설치' 혹은 '갖춤' 의 의미로 이해해도 되는 것일까.

his face 를,
사물을 대상으로 다루는 설치나 갖춤의 대상으로 말하지는 않는다.

구문 형태상으로 his face 는
set 동작의 대상(목적어)으로 보이지만

해석이 주는 의미에서 보아도 그렇고
다음 ② 같은 분석의 ⓒ 에서 구체화될 대상으로 나타나야 하므로

set 의 목적어(O)가 아니라
set 사역의 표적임을 알 수 있다.




①  10 years ago, I  set  about becoming an actor .
( 10년 전, 나는 배우가 되는 일을 시작했다 )


① 표현에서는 set 동작이 시작의 의미로 발화되었다.

그리고 형태적으로는
목적어가 없는 이른바 자동사로 구문에 작용하는 꼴이다.

그래서 모든 동사를 타동사화 해보는
다음과 같은 심층적 발화구조를 도입해서 분석하면

② 의 형태와 의미를 생각해 볼 수 있다.




②  I  set   myself   +   myself   is set+   it is   about becoming an actor
     ⓐ                              ⓑ                               ⓒ
  ( 나는 시작하게 한다 ) ( 나 자신은 시작하게 된다 )     ( 배우가 되는 것에 관해서이다 )


이와 같은 분석은, 심층 구조상으로
동사를 반드시 목적 대상에게 동작을 끼치게 하는 '사역성' 으로 타동사화 하고

주절 주어의 능동적 행위(ⓐ)에 따른 피역 대상의 수동(피동)적 처해짐(ⓑ)과
그것이 처한 상황에 대한 부가 서술(ⓒ)이

영어 구문 형성의 기저 현상임을 받아들이고 시도한 것으로써
이것은 다음과 같이 정리된다.


주어의 능동적 행위에 따른 의도대상의 수동(피동)화 현상이 영어구문 형성의 심층
기반에 있고 '주어 + 동사 + 대상(심층적 피동대상) + 상황구체화' 형식으로 나열된다.


이와 같은 구조분석의 틀에서 보면
set 의 동작은 자기 자신을 대상으로 다루고 있고

대상인 자신(myself)은
set 의 상태가 되는 수동적 처해짐에 있게 되는데

배우가 되려면
그 현장에로의 노출과 훈련이 필요할 것이고

이 상태를 (about 의 작용개념)의 상태로 보고
② 에서와 같은 구문 형태의 어휘를 선택해서 표현한 것으로 볼 수 있다.

그렇다면
어떤 환경(-업종)에 자기 자신을 노출시키는 것을 착수(시작)로 봤을 때

set 동작은
해당 환경에 들어서게 하는 사역 동작으로 작용해야 할 것이다.




③  He  set  out to buy food .
( 그는 음식을 사러 출발했다 )
④  I will  set  up in business .
( 나는 장사를 시작할 것이다 )


③ 에서의 출발 역시 착수시키는 사역에서 나올 수 있고

④ 에서 장사의 시작 또한 그 환경으로의 들어섬, 즉
그러한 환경에서의 '갖춤 상태' 를 set 이 표현해주고 있기 때문에

set + up 은
착수(준비) + 을 말하고 있는 셈이다.

그렇다고 해도 착수라는 개념은
이런 유형의 표현에 적용될 뿐, 다음 구문에서는 적용되지 않는다.
3. Her manner immediately set everyone at their ease.
( 그녀의 태도는 즉시 모든 사람을 편하게 했다 )

본문 3. 표현은
어떤 환경으로의 들어섬으로 이해하기 좋은 구문이다.

동사 set 으로 모든 사람이 어떤 환경에 들어서게 됐음을 말한 후
at their ease 를 부가해

'그 어떤 환경이 이것이다' 라는 식으로 구체화했기 때문에
편안함에 들어서게 되었다는 해석이 만들어진다고 할 수 있다.

그런데 set 으로 대상을 들어서게 할 때
들어서게 되는 환경이 어떠한지가 부가되어 서술됨을 볼 수 있다.




①  The sun  set  in the west .
( 해는 서쪽으로 진다 )
②  A hen  sets  on eggs .
( 암탉이 알을 품는다 )
③  The tide  sets  in .
( 조수가 밀려들어 온다 )


① ② ③ 예문들도
그 나름의 환경에 주어가 들어서게 되었음이 잘 적용된 표현들인데

① 의 경우는 서쪽이라는 이,

② 에서도 닭의 알 위의 (on)가 부가 서술되었고

③ 의 표현에서도
해안가 쪽이라는 속성(in)이 환경으로 부가 서술되어 있음을 참고하면

본문 3. 으로 표현되어진 '편안함' 은
부가된 at their easy 에서 비롯된 것임을 알게 된다.

따라서 set 동작으로의 사역은
'해당 환경으로의 굳힘' 을 겨냥한 동작임이 타당해지기 때문에

set 이
'굳혀짐에 들어서게 함' 의 사역을 한다고 보고 다음을 보자.
4. Leave the concrete to set for a few hours.
( 콘크리트가 굳어지도록 몇 시간 동안 나둬라 )

본문 4. 의 표현에서 보면,

콘크리트를 자연 환경에 노출시키면
시간이 지남에 따라 굳어지기 마련인데 이 상황이 시작됐음을 set 이 말하고 있다.

그리고
시작 이후 굳어지는 결과를 낳을 것은 자명하므로

set 으로의 시작(착수), 즉 들어서게 함은
어떤 상황으로의 진척까지를 개념화해야 합당해진다.




①  Her remarks  sets  me thinking .
( 그녀의 말은 나를 생각에 잠기게 했다 )


① 의 표현에서 생각에 잠긴다는 우리말 의미를 음미하면
생각이라는 동작의 연속됨이 떠오르기 때문에

그런 동태의 지속됨(굳혀짐)을 그려볼 수 있어서
개연성이 더해진다.




②  I'v  set  myself to finish the job by the end of the month .
( 이달 말까지 그 일을 끝마치기로 스스로 정했다 )


② 의 구문 표현 역시도
자기 자신에게의 set 사역으로 to finish, 즉 끝내려는 시도에 들어서게 하고

그것의 굳혀짐을 담아 표현하고 있다.




③  His face has  set . ( 그의 표정이 굳어졌다 )



③ 의 구문에서도
그와 같은 굳혀짐 상황에 들어섰음을 set 이 표현하고 있으므로

이 구문들의 set 표현을 정리하면 다음과 같이 드러난다.



첫째, set 은 대상으로 하여금 들어서게 하는 진입의 동태 사역 어휘이다.

둘째, set 으로 표현되는 진입은 '해당 환경으로의 들어섬' 이다.

셋째, set 에 의한 들어섬은 굳혀짐의 결과를 가진다.
