do [ 실현-제시 ]                    펼침 드러냄을 보게 하다         유튜브-영상 : shots

글자크기

동사이면서 조동사(auxiliary)로 작용하는 do 는
모든 '일반동사' 를 대신해주는 대동사로 작용하며

우리말 '하다' 에 해당하는 대용어로의 역할기능을 보여준다.




①  I will  do  it  without fail.
  ( 틀림없이 그걸 하겠다 )
②  Any chair will  do.
  ( 아무 의자라도 괜찮다 )


① 의 구문에서는
'하다' 라는 우리말을 적용해도 의미 전달에 있어서 무리가 없지만

② 의 경우는
'어떤 의자라도 한다' 라는 애매한 해석을 하게 될 것이다.




③  She  did  me a favor. ( 그녀는 내게 호의를 베풀었다 )
④  You  do  me proud. ( 저의 영광입니다 - 그것으로 저는 만족합니다 )
⑤  He  did  his friend die. ( 그는 친구를 죽게 했다 )



③ 과 ④ 에서 do 동사는
각각 수여-구문(4형식), 부여-구문(5형식)을 이루는 동작을 했지만

⑤ 의 경우는 이른바 '사역 구문' 이므로
모든 문장형식을 취할 만큼 do 의 활용 범위는 넓다.

다음은 조동사로 쓰인 경우이다.




⑥  I  did  not go there. ( 거기에 가지 않았다 )
⑦  I  did         go there. ( 거길 가긴 갔으나,... )



⑥ 에서의 did 는 '부정' 을 위한 삽입이고
⑦ 의 did 가 '강조' 를 위한 삽입이면

not 의 유무로
부정과 강조가 결정되게 하는 do 의 역할개념이 있을 것이므로

do 동사(조동사)가 쓰인 구문들을 살펴보며
역할기능과 그에 따른 작용개념을 도출해보자.
1. What shall I do?
( 난 어떻게 하면 좋을까? )

do 는 우리말 단어 '하다' 로 받아들이면
대부분의 구문에서 통용되어 진다.

또 영어 do 와 우리말 '하다' 가 해보이는 역할기능과 의미는
상당한 유사함도 보인다.

본문 1. 에서의 do 는 조동사 뒤의 위치에 원형 동사(원형-분사)로 선택되어
동작 형태만을 전하는 역할기능을 보여주는데

의문사인 what 을
do 의 대상(-목적어)이 되게 해주는 평서문으로 바꾸어 보면

do + what 은 '무엇을 하다' 가 된다.

그런데, 우리말 동사인 '하다' 역시도
어떤 명사나 부사, 형용사 어간에 붙어서

(운동 + 하다) (잘 + 하다) (착 + 하다) 와 같이
서로 다른 동작으로 인지하게끔 하는 대용어 역할을 하는데

'하다' 동사 자체로는 특정의 동작을 하지 않고
대상의 '동작화' 에만 관여함을 볼 수 있다.

그러면 do 가 드러내는 '하다' 는 어떻게 작용하는 것일까.




①  She     is  washing in stream under the mountain.
  ( 그녀는 산 아래 냇물에서 빨래를 하고 있다 )
②  She  does  washing in stream under the mountain.
  ( 그녀는 산 아래 냇물에서 빨래를 한다 )


① 과 ② 구문에서
She 의 동작은 washing 에 있고 공통이지만

is 와 does 에 의해 의미가 달라져
서로는 '하는 중이다' 와 '한다' 라는 의미 차이가 난다.

그렇다면, is 와 does 가 통사적으로 같은 위치에서
같은 서술 역할을 수행하고 있다고 보면

is 가 '' 을 나타내므로
does 역시 '주어 동작의 어찌함' 으로 주어의 동작 행태를 서술하는 것이어야 한다
2. Do you read it?  -  Yes I do.
( 너 그거 읽니? - 응 읽어 )

이번에는 조동사로 쓰인 경우인데,
본문 2. 같은 유형의 구문을 대하면

의문문이나 부정문 혹은 강조 구문을 표현하기 위해서
조동사인 do 를 삽입하고 있는 것으로 이해하게 된다.

그렇다면 뒤따르는 응답 구문에서의 do 는 무엇인가.




①  Can  you read it?   ( 넌 그걸 읽을 수 있니? )
②  Do  you read it?   ( 넌 그걸 읽니? )
③  Are  you reading it?   ( 넌 그걸 읽는 중이니? )



비교를 위해서 나열해 본 ① ② ③ 의 구문들은
모두 통사적(-형태적) 구조가 같다.

① 구문의 can 조동사(견해사)와 대응되는 위치에
② 의 구문에서는 do 가 있으므로

본문 2. 에서의 do 는
일반 동사가 아니라 can 과 같은 기능어(조동사)로 쓰였음이 분명해진다.

그리고
조동사는 화자의 견해를 나타내주는 역할기능을 하므로

② 표현에서 질문과 답변의 do 역시
'견해사' 로의 역할기능을 해 보이고 있어야 한다.

다음은 명령문의 경우를 보자.
3. Don't talk,  only do.
( 말로만 말고 실천해라 )

본문 3. 표현은 명령문의 연속으로
앞은 부정의 명령이고, 뒤는 조건달린 명령이다.

따라서 do 에 적용되는 역할기능은
조동사(견해사)와 일반 동사로 달리 작용한다.

그런데, 앞뒤 모두가
'하다' 라는 하나의 의미로 작용해야 한다면

본문 3. 의 앞뒤에 작용하는 공통의 표현 요소가 있을 것이므로
해석어 '말로만 말고 행동에 옮겨라' 에서 찾아보자.


'행동에 옮겨라' 라는 우리말은 곧 '실현(실천)하라' 이다.


그래서 다음 ① 과 ② 에서 not 에 의해 부정되는 대상은 다를지라도
do 를 '실현시킴'이라는 의미로 작용하는 화자의 견해 동작으로 생각해보자.




①  do  +  not talk ( 실현 시켜라 + 말 멈춤 동작형태 )
②  don't  +         talk ( 실현 시키지 마라 + 말 동작형태 )



'실현시킴' 이라는 동작 상황을
'펼쳐 보이는 동작' 으로 보고 해석해도 유사한 의미가 전해지므로

only do 에서의 do 도
앞의 조동사 don't 처럼 어떤 행위를 실현시키는, 즉

펼쳐 보이는 동작 표현으로 작용하고 있다고 볼 수 있다.




③  We must  do  something about it.
  ( 어떻게든 해야만 한다 )
④  Something must be  done  about it.
  ( 뭔가는 그것에 근접되어져야 한다 )


구문 ③ 의 해석은
'어떻게든 해야만 한다' 라고 의역되어 있지만

이 구문을 어휘별 심층 의미로 직역하면
수동화된 ④ 구문처럼, '뭔가를 그것에 근접되어지게 해야 한다' 가 되는데

이것은 전치사 about 의 '' 를 말해주는 역할기능에 의해서
it 에 '거의 다가감' 의 상태가 표현되기 때문이다.

이에 따라 do 동사에 '펼쳐 보임' 을 적용해보면
something 을 it 에 근접시키는 동작 상황이 나타난다.

다음의 심층 구조적 분석에서 그 현상을 이해해 보자.




⑤  We must  do   something  +   something   must be done +  it is  about it
                              ⓐ                                                                ⓑ                       ⓒ
          ( 뭔가를 펼쳐 보여야 한다 )           ( 뭔가는 펼쳐 보여져야 된다 ) ( 그것은 it 에 근접됨이다 )


⑤ 에서와 같은 심층적 분석에서 보면
본문의 의역 요소들은 다음의 과정에 있다.


ⓑ 의 심층적 수동화 ( something 의 펼쳐 보임 당함 )
ⓒ 의 표층적 결과화 ( something 의 펼쳐 보임 됨 ) = ( it 에 근접되어짐 )


그렇지만, 이런 심층 구조적 의미 생성 작용들이
본문 같은 의역을 나올 수 있게 하는 요소들일지라도

do 의 역할기능인 '펼쳐 보임 동작' 만으로는
do 의 역할을 이해할 수는 없다.
4. He does a book  from English  into Korean.
( 그는 영어로 된 책을 한국어로 번역한다 )

from 은 '' 을 지정 선언한다.

심층적으로 '본래 여기(이것)에 있었음' 이라는 의미로
상태를 구체화해서 보충시키는 전치사(상태사)로 작용하기 때문이다.




ⓐ  He  does   a book +   a book   is done +    it is   from English...
 ( 책을 펼쳐짐(실현) 되게 한다 )   ( 책은 펼쳐짐(실현) 된다 )           ( 그것은 영어로부터.... )



from 의 '기점' 을 적용해보면
본문 4. 에서는 he 의 동작인 does 가

a book 의 내용을 영어로부터 한국어로 변역해서
드러나게 펼쳐 보이는 동작을 한 것이 되는데

영어로 쓰인 책을
한국어로 드러나게 하는 '펼침 보임 됨' 을 말한 것이면

do 는 '실현 시킴' 과
그것을 '제시' 해주는 역할 기능적 특성을 가지고 있는 셈이다.

이와 같은 특성은 다음의 구문에서도 느낄 수 있다.




①  He  did  only 5miles  a day.
  ( 그는 하루에 단 5마일만 나아갔다 )


① 구문에 대한 우리말 해석이
'하루에 5마일만 나아갔다' 이고, 올바른 번역이면

do 가
5마일을 대상(목적어)으로 선택했을 때는

실현시켜서 그 결과를 볼 수 있게 한 '실현 제시' 의 의미역이
do 에 있기 때문에 그럴 수 있었을 것으로 추론할 수 있다.

그래야만
5 miles 을 '나아감' 의 대상 수치로 적용할 수 있기 때문이다.




②    He  did  me a favor.      
  ( 그녀는 나에게 호의를 베풀었다 )

③  He did  me   +     me     was done    +    it was   a favor
( 펼침 드러냄을 보게 했다 )     ( 그 드러냄을 보게 됐다 ) ( 그것은 호의였다 )



② 는, 어휘 나열 형태에 따르면
간접목적어와 직접목적어가 있는 4형식의 '' 으로 분석된다.

그런데, '수여' 의 의미에 따르면
주고-받는 동작 형태를 do 가 해보인 셈이지만

이 수여 현상은 do 자체에 수여적 동작 속성이 있어서라기보다는
간접목적어인 me 의 심층적 수동 현상과

그 수동 상태를 구체화하기 위해 선택한
직접목적어 a favor 의 순차적 나열로 인해서 발생하는 꼴이다.

그렇다면 ② 에서 '호의' 가 베풀어지고, 그것을 받는(당함) 수혜 대상이 있으므로
do 가 수여적이게 되는 것은

펼침의 드러냄을 볼 수 있게 해주는 사역의 동작 속성이
그것을 가능하게 했기 때문으로 볼 수 있다.

이 사역 동작 속성을 ④ 에 적용해보자.




④  I  did  the town  yesterday.
  ( 난 어제 그 도시를 구경했다 )

⑤  I  did      myself        +     myself    was done      +    it was   the town
( 펼침 드러냄을 보게 했다 ) ( 그 드러냄을 보게 됐다 ) ( 보게 된 것은 그 도시였다 )



④ 는 '하다' 라는 우리말 의미를 적용했다면
수용할 수 없는 해석의 구문이지만

'펼침 드러냄을 보게 하다' 라는 심층 사역을 한 것으로 보면
도시의 이모저모를 구경했다는 것이 가능해진다.

그래도 무엇이 드러나는가? 라는 문제가 남는다.




⑥  I'v been  done  for speeding.
  ( 난 속도위반으로 걸렸다 )


⑥ 의 표현을
'속도 위반으로 걸렸다' 라는 우리말 해석에 따라 분석하면

전치사 for 는
I've been done 된 것에 대해 '' 을 하겠다는 선언인 셈이다.

그래서 speeding 때문이라고 하는
해명 구절을 이끌게 되는 specification(-의미 지정) 으로 작용한다.

그러면 '속도 위반으로 걸렸다' 중에서
'속도' 는 speeding 으로 표현되었으므로

남은 부분은
'위반으로 걸렸다' 라는 우리말 해석의 출처를 찾아야 한다.

앞에서 '드러냄' 동작을 도출했는데
이 구문에서는 무엇이 do 에 의한 펼침 드러냄의 대상인가.

먼저 이 ⑥ 수동 구문을
능동 구문화 해보면 이렇게 된다.




⑦  I'v been  done  by the police for speeding.
  ( 난 속도위반으로 경찰에 적발되었다 )
⑧  The police has  done  me for speeding.
  ( 경찰은 속도위반으로 나를 적발했다 )


영어 표현에는 주절의 능동적 행위에 따라 종속절의 수동화 현상이 있게 되는데
이것을 적용하면 이렇게 된다.



⑨  The police has  done   me  +   me  have  been done by the police  +    it is   for speeding
        ( 경찰은 내게 드러냄을 했다 )         ( 난 경찰에게 드러냄 당했다 ) ( 그것(-드러냄)은 속도 때문이다 )



⑨ 의 분석에 따르면
수동화된 me have been done by the police 가

화자가 표현하고자 하는 일차 정보가 되어
표층으로 드러나면 ⑦ 이 된다.

그런데 do 동작으로 표현된 The police 의 적발 행위, 즉
'드러냄' 은 무엇인가.

뒤에서 for speeding 으로 해명했으므로
'속도 위반에 대한 적발' 이 분명하다면

do 에 의해 드러난 '펼침 드러냄 동작' 은
경찰의 적발 행위일 것이다.

위반이 저질러 질 때는 위반 사항으로 드러나지는 않는다.
그것이 경찰의 적발 행위에 의해 비로소 드러나게 되는데

이렇게 드러내는 행위가
해당자인 me 에게 가한 경찰의 행위이고 보면

do 는
'펼침의 결과를 노출' 시키는 '실현 제시' 를 하는 것이 된다.

또 다른 구문을 통해서
이 구문에서 드러낸 '위반 사실' 과 같은 '드러나게 되는' 내용물의 형태를 알아보자.
5. Joseph speaks Japanese better than I do English.
( 요셉은 내가 영어를 하는 것보다 일본말을 더 잘한다 )

①  Joseph speaks  Japanese better than I  do  English .
  ( 요셉은 내가 영어를 하는 것보다 일본말을 더 잘한다 )
②  Joseph wanted to speak Japanese often, and he  did  often.  = spoke Japanese
  ( 요셉은 가끔 일본어를 말하기 원했고, 그리고 가끔은 했다 )


① 구문에서 do 는
형태상으로 주절 동사 speak 의 중복을 피하는 대용어로 쓰였다.

이것이 대동사 기능인데
② 구문의 경우 did 는 대동사로 설명이 가능한가.

만약 가능하다면 주절 동사 want 의 대동사인가
아니면 (to)speak 의 대동사인가.

do 가 단지 동사만을 대신한다면
want 와 speak 중 어느 하나여야만 한다.




③  He  wanted   +   ( himself )  to speak Japanese often   +    and he did  (it)  often
       ( 요셉은 자기가 가끔 일본어를 말하기 원했고 ) ( 그리고 가끔은 그 원하는 바를 했다 )


and 이하의 did 가 want   의 대동사로 작용했으면 다음 ④ 와 같이 될 것이고
and 이하의 did 가 speak 의 대동사로 작용했으면 다음 ⑤ 와 같이 될 것이다.




④  Joseph  wanted  +   ( himself )  to speak Japanese often.
       ( he )      ( did )                          ( it )


⑤  ( himself )   +   to speak  Japanese.
     ( he )              ( did )      ( it )


did 가 ④ 같은 주절 구문 want 에 작용하는 '대동사' 이면
wanted = did 이어야 하고

⑤ 의 종속 구문 to speak에 작용하는 '대동사' 라면
(to) speak = did 가 되어야만 하는데

그렇지만
③ 표현의 심층적 해석인 '원하는 바를 했다' 를 받아들이고 보면

he did = he spoke Japanese 라는 표현임이 분명하므로
동사가 아니라 '동사구' 를 대신하고 있어서 다음의 구조로 생각해야 한다.




⑥  ( himself )   +   to speak  Japanese.
     ( he )        ( did )      ( it )


이런 현상은 do 가
동사만을 대신한다는 '대동사' 라는 기능으로는 설명될 수 없으며

대신하는 요소가 있다면
그것은 '동사구' 모두를 대신하는 '대용어' 로 봐야한다.

그런데 ③ 에서의 did 는
wanted + (himself) to speak Japanese 모두를 동사구로 하는 대용은 하지 못하는데

이것 또한 do 의 특성과 무관할 수 없다.

문장 전체에서 보면, wanted 는 말하는 시점에서의 '바램' 인 마음의 상태로써
이미 드러낸 과거형 동작이지만

to speak 는 특정의 시제가 아닌 해당 시점형 동태라서
말을 할 것이거나, 말을 했을 것으로 적용시킬 수 있는 to-부정사이다.

화자는 먼저 '바램(want)' 을 말하고서
그 바램이 어떤 동작(speak)인가를 말했다.

그리고 아직 드러내지 않은 동작 형태를
did 라는 동사로 말해 and 접속어로 연결했는데

만약 동사 did 가 wanted 이하를 대용하고 있다면
and 로 접속된 구문 ② 에서는 wanted and did 가 되므로

'원했고 그리고 그는 원했다' 라는 식의 중복 표현이 되어
의미상 비문이 된다.

왜냐하면
'and' 접속사의 사용에서는 '순차성' 을 발견할 수 있는데

앞뒤 사건이 중복될 수는 없고, 또 시간의 흐름상으로 '역순' 되지도 않는
순차적으로 표현되는 것을 볼 수 있기 때문이다

그러므로 do 는
to speak 이하를 대신하는 것이 분명하고

do 의 역할기능이, 아직 드러내지 않은 대상을 실현시켜 보게 한다는
화자의 견해 삽입임을 말해 준다고 할 수 있다.
6. He did the garage.
( 그는 차고를 청소했다 )

앞에서 I did the town 을 '구경했다' 라고 해석할 때
'펼침 드러냄을 보게 한다' 라는 개념이 작용했다고 보았고

다음 ① 구문은, 같은 개념이 적용되어 있다.




①  They  did  the town yesterday.
  ( 그들은 어제 그 도시를 구경했다 )


본문 6. 에도 같은 개념이 적용되어야 하는데
The garage 는 차가 들어갈 수 있는 공간이다.

그래서
차가 들어갈 수 있도록 이것저것을 치웠다는 것이 된다.

그 결과 차고로서의 '형태 드러냄' 이 나타났을 것이고
그것을 대상 삼아 보게 했다는 사역 동작을 did 로 표현해 준 것이므로

do 의 역할 기능은 '제시적' 일 수밖에 없다.

다음 ② 구문처럼
in yellow 를 덧붙여 말하면 '차고를 노랗게 칠했다' 뜻이 되는 것도

노랗게 칠이 된 드러냄의 대상을
볼 수 있게 해 두었다는 심층적 의미가 있어야 가능할 것이다.



②  They  did  the garage  in yellow.
  ( 그는 차고를 노랗게 칠해 놨다 )




7. I don't want you to stay.
( 나는 네가 머물러 있기를 원치 않는다 )

not 이 개입된 다음 ① ② 의 예문에서
do 에 작용하는 심층개념의 개연성을 넓혀보자.




①  I  don't    want you to stay. ( 원하는 마음 드러냄을 보이지 않음 )
②  I  do  not want you to stay. ( 원치 않는 마음 드러냄을 보임 )



① ② 구문의 의미 차이는 무엇인가.

① 표현의 악센트는 don't 에 있고
② 표현의 악센트는 not 에 있다.

이것은
화자가 부정하려는 내용이 무엇인가를 알려주는 방법일 것이다.

① 의 부정 대상은 do 자체이다.
그러나 ② 의 부정 대상은 want 일수 밖에 없다.

not 부정은 뒤따라 나타나는 어휘를 부정할 뿐이라서
앞선 어휘는 영향을 주지 않는다.

그래서
do 자체를 부정하려면 don't 같은 축약이 필요해진다.

그렇다면
의미적으로는 어떤 차이가 있을까.

두 구문 모두에서 want 는 원형으로 쓰였다.
다시 말해서

'원하기' 라는 속성으로써의 동작 형태만을 표현해주는 것이라서
화자의 주된 전달 의사는 do 와 not 에 있다.

① 구문의 경우는
'원하는 마음의 드러냄' 을 보게 하지 않는다는 표현인 반면에

② 구문의 경우는
'원하지 않는 마음의 드러냄' 을 보게 한다는 차이로 변별된다.

여기서 '원하는 마음(want)' 과 '원하지 않는 마음(not want)' 은
겉으로 드러내기 이전의 마음 먹음이다.

이미 내적으로 결정지어진 행위이다.

이것이 do 가 드러나게 하는 내용물이고
다음의 부가 의문문을 보면 알 수 있다.




③  He whispered Tom not to be afraid,   didn't  he?
  ( 그는 탐에게 두려워하지 말라고 속삭였지, 아니니? )


부가 의문절 didn't he? 를
'그러지 않았니?' 라고 하게 되는 것도

주절 주어인 he 가
속삭이는 모습을 드러나게 하지 않았느냐는 반문이기 때문이다.

결국 do 는 '어떤 꼴의 노출시킴' 이 되는데
이렇게 부가의문문에서 추정해 볼 수 있듯이

didn't(-드러내지 않았느냐?) 를 먼저 말하면
그 뒤에 복원되어질 수밖에 없는 생략 내용은

앞에서 말한 내용인
이미 저질러진, 결정지어진 행위가 될 것이다.

그러면
'저지른' 혹은 '이미 결정지어진 행위' 는 어떠한 모양새일까.
8. Things done cannot be undone.
( 이미 저지른 일은 무를 수 없다 )

이미 어떠한 행위가 'things' 에 가해진 상태이고
그래서 그 행위는 이미 끝나 있음을 알 수 있다.

달리 생각하면
'엎질러진 물은 주어 담을 수 없다' 와 같은 뜻을 전하는 꼴이다.

여기서의 done 역시
어떤 행위를 가해 이미 끝나있음을 말해야 하는 것이므로

우리말 '펼침 드러냄' 에 해당시켜 보면
선택의 여지 없는 수용의 결과가 된다.

어떤 반복되는 지겨운 현상에 대해서
그 상황을 수용해야 하는 듯한 영국식 표현 ① 에 적용해 보자.


①  That   does  me. ( 그것은 당해낼 수가 없다 )            

표현 ① 만으로는 무엇을 말하는지 알 수 없지만
That 으로 지정한 어떤 상황이 있었음을 가상하고 이해하면 이렇다.



②  That  does   itself   +    itself     is done   +   it is  me         
 ( 펼침 드러냄을 보게 한다 ) ( 펼쳐짐을 보게 된다 ) ( 보게 되는 것은 me 다 )


② 의 분석에서처럼 목적어(-3형식의 표적 대상)인 me 가
심층적으로 처해지는 수동화를 구체화하는 대상으로 쓰였음을 받아들이면

me 는
어떤 펼침 드러냄을 보아야만 하는 표현 구문이 된다.

예문 ① 에서의 해석처럼
'그것은 당해낼 수 없다' 같은 표층적 의역이 가능한 것도

동사 do 의 표적 대상인 me 가
That 으로 단정된 어떤 펼침 드러냄을 감당해야만 하기 때문에

드러난 그대로를
선택의 여지 없이 수용하는 듯한 표현이라 그럴 것이다.



③  Another late train? ( 또 지각이야? )

        --  that   does  me. ( 진짜 질린다 )


④  They changed the deadline again? ( 마감일 또 바뀌었어? )

        --  that   does  me. ( 진짜 너무해 )



예문 ③ ④ 에서도 보면
그 같은 ‘상황 펼쳐짐 보게 됨’ 을 수용할 수밖에 없을 때의 댓구 표현이고

다음 ⑤ 표현도 욕 다음의 발화라서
‘해치움’같은 부정적인 의사표시로 나타난다고 볼 수 있다.



⑤  Damn you, I'll   do  +   for you . ( 망할 놈! 죽여 버리겠어 )
 ( 펼쳐(-해치움) 드러냄을 보게 하고 말겠어 + 널 두고 하는 말이다)


물론, 모든 표현에서 부정적이지는 않다.
예문 ⑥ 표현에서는 긍정적인 수용으로 표현되었는데

그럴지라도 선택의 여지를 두지 않는 듯한
더이상은 어쩔 수 없다는 체념적 수용임을 알 수 있다.



⑥  How about we meet at 3 p.m. ( 3시에 만나는 것 어떼요? )

        --  that   does  me. ( 그 시간 괜찮아요 )





9. Fifty dollars will do me fine.
( 50달라면 됩니다 )

본문 9. 는 부여-구문이다.
따라서 me 는 fine 상태를 부여받게 되는 의미표현이다.

주절의 동사 do 가
me 를 상대로 펼침 드러냄을 보게 할 것이라고 말한 것이면

심층 종속절에서의 의미는 ① 과 같아야 한다.




①  Fifty dollars will  do   me   +   me    shall be done       +   it is   fine            
     ( 돈은 펼침(만족?) 드러냄을 보게 한다 )   ( 나의 드러남은 비춰질 게다 ) ( 그건 fine 상태로 이다 )



주절에서 do 의 발화로 표현한 '드러냄 보여줌' 은
심층 종속부에서 찾을 수 있다.

'종속절의 수동적 처해짐' 에 따르면 '드러나게 됨' 은 be done 으로 표현되는데
be done 이라는 수동적 처해짐은 상황 부가부에서 구체화되기 때문에

드러냄이 비춰지는 모양새는
it(me shall be done) is fine 에서 확인되는 셈이다.

이 작용들이, 본문의 해석에서
'50불은 나를 fine 상태가 되게 해줄 것이다' 라는 직역을 가능하게 한 것으로 보고

do 를 '펼침 드러냄을 보게 한다' 라는 사역 동작으로 수용해서
다음의 구문들도 이해해 보자.




②  He  did  a Hamlet . ( 그는 햄릿을 흉내 냈다 )
③  He  did      Hamlet . ( 그는 햄릿 역할을 했다 )



두 예문에서의 의미 차이는
와 무관사에서 비롯된다.

② 는 a 의 수식으로 인해 햄릿이라는 유형으로 '펼침 드러냄을 보게 했다' 가 되어
'흉내' 라는 근거가 되고

③ 에서의 무관사는 '태생적인 자기 유형'을 말하기 때문에
햄릿 그 자체는 펼침 드러냄의 역할 대상이기에 충분하다.


do 에 대한 예문들을 통해서 드러난 특성들을 정리해 보자.



첫째, do 는 실현된 어떠함을 드러내서 보게 하는 이중 동작을 한다.

둘째, do 의 '드러냄을 보게 함' 은 주어의 사역도, 화자의 견해도 되어준다.

셋째, do 에 의한 드러냄의 실상은 펼침의 결과물이다.
