run [ 곧장-이끔 ]                멈춤 없이 이끌어지게 한다

http://www.grammar.kr : 다음사전
글자크기

동사 run 에 대한 해석의 우리말 의미는 '달리다' 로 대표된다.




①  He  runs  as fast as you .
  ( 그는 너만큼 빨리 뛴다 )


그런데
모든 상황에서 run 의 쓰임을 '달리다' 로 해석할 수만은 없다.




②  I'll  run  over to the supermarket .
  ( 수퍼마켓에 잠깐 들러야겠다 )


수퍼마켓에 잠시 들름을
뛰어가는 표현으로 말하지는 않을 것이기 때문이다.

그리고 into 로 상황이 구체화되는 다음의 표현에서는
'우연' 의 결과표현을 하기도 한다.




③  I  ran  into her . ( 나는 그녀를 우연히 만났다 )



그뿐 아니다.
run 이 '경영' 을 말하고, '출마' 를 표현하기도 한다.




④  I just  run  it for the owner .
  ( 주인을 대신해 경영하고 있을 뿐이다 )
⑤  He  runs  for president .
  ( 그는 대통령에 출마한다 )


또 다음의 표현에서는
왜 신호 무시로 해석되는지를 이해할 수 있어야 한다.




⑥  Don't  run  the light .
  ( 신호를 무시하지 마라 )


run 이 쓰이는 몇몇 구문들을 분석하면서
이 어휘가 드러내는 역할기능과 그에 따른 작용개념을 도출해보자.
1. The child ran to his mother for protection.
( 아이는 엄마 품으로 달려갔다 )

동사 run 에 해당하는 우리말 사전적 어휘로는
먼저 '달리다' 를 수용하게 된다.

본문 1. 표현에서도
아이의 움직임은 엄마에게로의 빠른 이동으로 느낄 수 있기 때문이다.

그런데
이 run 으로 표현된 달리는 동작의 속도는 어느 정도나 될까.

이 어휘 run 자체만으로는 속도를 규정 지울 수 없지만
걷기 이상의 속도감을 달리는 동작으로 보면

run 은 어느 정도의 속도가 있는 동태를 표현을 하는 것으로 느껴지므로
걷기 이상의 활동성을 드러내는 표현으로 볼 수 있다.




①  He  runs  as fast as you .
  ( 그는 너만큼 빠르다 )


① 에서도 fast 라는 수식어가
run 이 드러내는 속도의 어떠함을 가늠하게 해주기는 하지만

어느 정도나 빠른지의 절대치는 드러내 주지는 않는다.

만약 you 가 아주 느리게 달리는 사람이면
he 도 그만큼만 빠르다는 것이 되므로

여기서도 속도감은
상대적 비교치일 뿐 절대적이지 못하다.




②  It is five minutes' run  to the station .
  ( 역까지는 달려서 5분 거리다 )
③  He came here at a  run .
  ( 그는 한걸음에 여기 왔다 )


명사로 쓰인 ② ③ 표현들에서도 역시
run 이 육상 선수의 달리기를 말한 것은 아닐 것이므로

잰걸음 이상의 활발한 움직임으로 받아들이고
다음을 보자.
2. Long toes run in family?
( 긴 발가락은 집안 내력인가요? )

본문 2. 표현에서는 run 이 '내력' 으로 해석되었는데

내력이라는 의미를
'달리다' 와 같은 동작 개념으로 표현하지는 않았을 것이다.

내력의 의미는 집안에서 이어져 내려오는 어떤 현상인데
이와 같은 유전 현상을 두고 속도(-달리다와 같은)를 말할 수는 없을 것이다.

속도와는 무관한 계속되는 내려옴, 즉
끊이지 않는 이어짐을 표현한 것이 자명하므로

run 은
계속되고 있는(-멈춤 없는-) 동태 표현으로 이해될 수 있다.




①  She  runs  a fever .
  ( 그녀는 열이 난다 )
②  The book  runs  10 dollars .
  ( 그 책은 10불 한다 )
③  The lake  runs  dry .
  ( 그 호수는 말라가고 있다 )


제시된 세 예문 모두에서도 어떤 빠른 움직임은 전혀 느낄 수 없다.

① 의 경우는 열의 지속 현상이 있을 것이고
② 의 경우는 책의 거래되고 있는 시세를 말한 것이다.

③ 의 구문에서도 말라가는 현상은
'서서히 진행되지만 멈춤 없는' 지속 상태에서 나타날 것이므로

run 동작에 대한 속도감을 달리 생각해보자.
3. I just run it for the owner
( 주인을 대신해서 그것을 경영하고 있을 뿐이다 )

본문 3. 에서는
동사 run 이 '경영' 을 표현해주고 있는데

멈춤 없는 활동성이 run 의 역할기능으로 작용하는 것이라서
이 표현을 가능하게 한다고 보기에는 이 개념만으로는 부족하다.

우리말 '뛴다' 라는 의미로 경영을 표현해 줄 수 있을지라도
본문에는 그런 개념이 적용되지 않음을 느끼게 된다.

'뛰다(-달리다)' 라는 우리말 동작 의미를 적용하면
run 은 문법적으로 자기 동작, 즉 자동사로 기능하게 된다.

본문 3. 은 it 을
경영시키는, 즉 run 하게 하는 타동사적 해석을 해야만 하므로

이 현상을 이해하기 위해서는
① 과 같은 심층적 구문분석 틀에서의 이해가 필요해진다.




①  I  run  it    +  it   is run+   it is   for the owner
       ⓐ                   ⓑ                     ⓒ
  ( 그것을 이끌어간다 ) ( 그것은 이끌어 진다 ) ( owner 를 염두에 두고 이다 )


이와 같은 분석은, 심층 구조상으로
동사를 반드시 목적 대상에게 동작을 끼치게 하는 '사역성' 으로 타동사화 하고

주절 주어의 능동적 행위(ⓐ)에 따른 피역 대상의 수동(피동)적 처해짐(ⓑ)과
그것이 처한 상황에 대한 부가 서술(ⓒ)이

영어 구문 형성의 기저 현상임을 받아들이고 시도한 것으로써
이것은 다음과 같이 정리된다.


주어의 능동적 행위에 따른 의도대상의 수동(피동)화 현상이 영어구문 형성의 심층
기반에 있고 '주어 + 동사 + 대상(심층적 피동대상) + 상황구체화' 형식으로 나열된다.

사업을 경영하는 사람이
그 사업을 '끌어간다' 고 표현해도 의미가 통하고

경영자의 활동성을 동반해 반영하면
'이끌어 간다' 라는 동작이 될 수 있으므로

① 의 분석과 같은 사역성을 부여해보면
'이끌어 가게 한다' 하는 심층적 의미를 도출해 낼 수 있다.

그런데 it(경영) 은
'사역' 대상이 아니라 '피역' 대상이므로 달리 분석되어야 한다.




②  I  run  myself     +   myself   is run  +   it is   it(경영) for the owner
       ⓐ                             ⓑ                         ⓒ
  ( 그것을 이끌어간다 )   ( 그것은 이끌어 진다 ) ( 그것은 owner 를 염두에 두고 하는 경영이다 )


② 구조에서
상황 구체화 어구(ⓒ)의 It  for the owner 는 It is for the owner 이고

전치사 for 는
‘’ 로 해명을 하는 ‘상태사’ 로 보고 분석한 것이다.

다음 ③ 표현은 3형식의 이지만
④ 의 심층구조에서처럼 주어 자신을 사역시키는 것으로 분석되어야 하므로

본문 3. 도 ② 와 같은 이해가 필요해진다.




③  He  runs  the engine . ( 그는 엔진을 작동시킨다 )


④  He  runs   himself   +   himself   is run    +    it is   the engine
            ⓐ                               ⓑ                   ⓒ
  ( 이끌어 가게 한다 )     ( 이끌어 가게 된다 )        ( 그것은 엔진 작동이다 )




4. She runs her fingers through her hair.
( 손가락으로 머리카락을 빗는다 )

본문 4. 는 사역 작용을 잘 드러내 주는 구문이다.

심층적으로는 그녀 자신이 사역 대상이지만
표층적 구조에 적용해보아도 her fingers 는 사역 당하는 것으로 이해할 수 있다.




①  She  runs  her fingers    +  her fingers   are run  +   it is   through her hair
                 ⓐ                                                ⓑ                     ⓒ
     ( 손가락을 이끌어가게 한다 )    ( 손가락을 이끌어가게 한다 )  ( 머리카락을 관통하면서이다 )



손가락을 머릿결 따라 움직이게 하는 것은 주어 자신이기 때문에
이런 형태로의 분석 또한 가능하므로

run 을 이끌어 가게 하는 사역 동작으로 작용하는 것으로 볼 수 있고
다음 구문들 또한 이해할 수 있게 된다.




②  Ivy  runs  up the walls of the house .
  ( 담쟁이덩굴이 집의 벽을 타고 오른다 )
③  Emotions  run  deep within our race .
  ( 우리의 집단 내에서 감정은 깊어진다 )


② 에서는 담장을 타고 오르며 자라는 덩굴에서 속도감을 찾을 수 없고
담장을 타고 오르게 하는 '이끌어짐' 만이 있다.

③ 에서는 감정(Emotions)이 주어이고
자기 사역으로 깊이로의 '이끌어감' 을 하겠지만

깊이의 끝은 드러나지 않는, 즉
멈춤 없는 상태(동태) 표현이 되어줄 뿐이다.
5. Don't run the light.
( 신호를 무시하지 마라 )

본문 5. 의 해석을 보면
run 이 자기 사역을 해야 하는 동작임을 알 수 있게 된다.

차를 운전하는 사람을 향해 하는 발언이면
신호를 무시하고 차를 이끌어 가고 있는 운전자(주어)를 향해 외치는 명령이기 때문에

Don’t run 에서 '이끌어감' 이행 동작의 멈춤은
운전자 자신이 해야 하므로 자기 사역이다.

그런데, 왜 이 구문이
신호를 무시하지 말라는 표현이 되는 것일까.

the light 가 run 동작에 의해 이끌어 진다로 해석된다면
신호를 무시하는 표현으로 받아들일 수 없을 것이다.

그렇지만, 앞 본문 2. 에서 도출해 둔 계속되고 있는(멈춤 없는) 동태 표현을 수용하면
'멈추지 않는 이끌어짐'을 run 이 말하는 것이므로

신호등 앞에서 신호에 따르지 않고 계속하는 운행이 곧
신호를 무시하는 상황임을 그려볼 수 있다.

다음의 구문들은
그와 같은 '멈춤 없는 이끌어짐' 이 표현되는 좋은 예문들이다.




①  Blood is  runing  from my nose .
  ( 코피가 나고 있다 )

②  He  ran  his car into a tree .
  ( 차로 나무를 들이 받았다 )

③  An angry murmur  ran  through the crowd .
  ( 성난 중얼거림이 군중들 사이로 퍼져 나갔다 )


지금까지 살펴본 run 동작에 작용하고 있는 역할기능과
그에 따른 작용개념을 다음과 같이 정리할 수 있다.



첫째, run 동작은 '이끌어지게 하는' 운영 행위이다.

둘째, run 으로 표현되는 운영 행위는 연속성을 가진다.

셋째, run 동작이 낳은 상황은 멈춤 없이 전개된 결과이다.

작용개념 : 멈춤 없이 이끌어지게 한다
